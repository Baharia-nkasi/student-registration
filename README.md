# Student Registration System

## 📌 Description
This is a simple Student Registration System developed using PHP and MySQL (XAMPP). It allows users to register, login, and manage student records.

## 🚀 Features
- User Registration
- User Login & Logout
- CRUD Operations (Create, Read, Update, Delete)
- Session Management
- Password Security

## 🛠 Technologies Used
- PHP
- MySQL (XAMPP)
- HTML
- Git & GitHub

## ⚙️ How to Run
1. Install XAMPP
2. Place project in htdocs folder
3. Start Apache & MySQL
4. Import database
5. Open browser:
   http://localhost/student-registration
